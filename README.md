## Contenido del repositorio

- `presentacion.qmd`: archivo fuente de la presentación.
- `styles.css`: estilos visuales.
- `video-loop.html`: reproducción automática de los videos.
- `videos/`: archivos MP4 utilizados en la presentación.
- `imagenes/`: logo y recursos gráficos.
- `docs/`: presentación HTML renderizada para GitHub Pages.
- `_quarto.yml`: configuración del proyecto Quarto.
- `generar_videos.R`: código utilizado para generar las animaciones.

## Recompilar la presentación

1. Descargar y descomprimir el repositorio.
2. Instalar Quarto.
3. Abrir una terminal dentro de la carpeta del proyecto.
4. Ejecutar:

```bash
quarto render
